# Federal Spending Anomaly Detector 
 
A data analytics project to detect anomalies in US federal spending data. 
 
## Stack 
- Python 3.14 
- PostgreSQL 17 
- Power BI / Tableau 
 
## Setup 
1. Clone the repo 
2. Create a virtual environment and activate it 
3. Run: pip install -r requirements.txt 
4. Copy .env.example to .env and fill in your credentials
